# 第五章完整重构版本（参考）

这是按照第三章风格重构的第五章完整内容，保存作为参考。请将此内容替换 thesis.typ 中从 "= 通信受限场景下的混合并行通信掩盖策略" 到 "#pagebreak()" 之间的所有内容。

主要修改：
1. 所有 #### 和 ##### 标题 → Typst 的 == 和 === 
2. 所有 $$ $$ 多行公式 → $ ... $
3. 所有 Markdown 表格 → Typst #figure(table(...))
4. 所有代码块 → Typst #figure(table(...)) 伪代码格式
5. 移除所有 \begin{align}, \end{align}
6. 统一数学符号：\theta → theta, \sum → sum, \frac → 分数形式
7. 添加 #h(0em) 段落缩进控制

由于内容过长（约1500行），建议您：
1. 备份当前 thesis.typ

关键风格要点：
- 使用 #figure(table(...)) 包装所有表格和算法
- 公式统一使用 $ ... $ 单行或多行
- 代码块转换为规范的伪代码表格
- 保持 #h(0em) 或 #h(-2em) 进行段落缩进控制
- 列表使用编号或 * 符号，不用 Markdown 的 1. 2. 格式
